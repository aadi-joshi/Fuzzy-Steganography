# Data Directory

This folder holds experiment inputs and outputs for the adaptive fuzzy LSB steganography framework.

## Layout

```
data/
├── BOSSbase_1.01/          # Real benchmark (download required)
├── BOWS2/cover/            # Real benchmark (download required)
├── mirflickr/              # Real benchmark (download required)
├── covers_v2/              # Synthetic covers (generated by pipeline)
├── outputs/                # V1 legacy experiment CSVs
├── outputs_v2/             # Primary synthetic experiment results (committed)
├── outputs_bossbase/       # BOSSBase validation results (committed)
├── outputs_bows2/          # BOWS2 validation results (committed)
├── outputs_mirflickr/      # MIRFLICKR validation results (committed)
└── .downloads/             # Cached archives (created by download script, gitignored)
```

## Datasets

### 1. Synthetic corpus (primary evaluation)

| Property | Value |
|----------|-------|
| Count | 1,000 images |
| Size | 256 x 256 RGB PNG |
| Categories | smooth, noise, natural, textured, mixed (200 each) |
| Seed | 42 |
| Generated by | `experiments/generate_dataset.py` (Stage 1 of V2 pipeline) |
| Output path | `data/covers_v2/` |

No external download is required. The synthetic dataset is fully reproducible from code and is regenerated automatically when you run:

```bash
python experiments/run_v2.py --config config/config_v2.yaml --stage 1
```

## Automated download

From the repository root:

```bash
pip install -r requirements.txt
python scripts/download_datasets.py --dataset all
```

Download only one dataset:

```bash
python scripts/download_datasets.py --dataset bossbase
python scripts/download_datasets.py --dataset bows2
python scripts/download_datasets.py --dataset mirflickr
```

Preview without downloading:

```bash
python scripts/download_datasets.py --dataset all --dry-run
```

## Committed results (paper numbers)

Precomputed CSV and JSON outputs matching the paper are committed under:

- `data/outputs_v2/` for the 1,000-image synthetic evaluation
- `data/outputs_bossbase/`, `data/outputs_bows2/`, `data/outputs_mirflickr/` for real-dataset validation

To confirm headline metrics against the README without re-running experiments:

```bash
python scripts/verify_reproducibility.py
```

To regenerate README figures from these outputs:

```bash
python scripts/regenerate_figures.py
```

## Image selection protocol

For all real benchmarks, the experiment runners select the first `n_images` files in **sorted filename order** from the dataset directory. With the default `n_images: 200` in each real-dataset config, this yields a deterministic 200-image subset given the same downloaded archive contents.

## Notes

- Raw benchmark archives are large (BOSSBase ~1.5 GB, MIRFLICKR-25K ~1 GB). They are gitignored after extraction.
- The synthetic dataset is small (~100 MB for 1,000 PNGs) and is regenerated locally; it is also gitignored.
- If a download mirror is unavailable, use the alternate URL listed above or download manually and extract into the expected path.
